# C# Battle Arena Simulator

A console-based C# game project demonstrating object-oriented programming concepts such as inheritance, polymorphism, abstraction, encapsulation, and interfaces.

## Features
- Multiple character classes
- Turn-based battle system
- Special attacks
- Random enemy generation
- Replay option
- Clean OOP structure

## Technologies Used
- C#
- .NET 8 Console App
- Object-Oriented Programming

## Project Structure
- Interfaces
- Models
- Services
- Program.cs

## How to Run
1. Open the project in Visual Studio
2. Build and run the application
3. Choose a character
4. Battle against a random enemy

## OOP Concepts Demonstrated
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Interface implementation
